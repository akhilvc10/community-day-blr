Event Information

This is a Bengaluru based group for anyone interested in Amazon Web Services, Cloud Computing, AWS design implementation and servicing, EC2, S3, High Scalability Computing, Real Life uses of AWS and cloud servers, SaaS, IAAS, PAAS and many more.

  • Explore AWS tools and offerings by various organizations
  • Win AWS Goodies
